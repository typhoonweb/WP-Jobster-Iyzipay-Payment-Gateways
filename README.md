Iyzipay payment method attempt for Wordpress WP Jobster theme works inconsistent.


# Requirements

* [**Iyzipay-php**] (https://github.com/iyzico/iyzipay-php)
* [**WP Jopster**] (https://wpjobster.com/wp-content/uploads/2020/07/wpjobster-sample-gateway-2.7-ts1592932967.zip)


# Installation
Save the downloaded files in the wordpress plugin folder.


# Activate
From the admin panel, set it on Jobster > Payment Gateways > Iyzipay page.
